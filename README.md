# Interactive Web Resume (JA)

React + TypeScriptスタックで構築された、モダンでインタラクティブ、かつ完全にレスポンシブなウェブ履歴書です。このプロジェクトは、パフォーマンス、クリーンなコード、および快適なユーザーインターフェイスに重点を置いて開発されました。

## 技術スタック

*   Framework: React 18 (Strict Mode)
*   Language: TypeScript
*   Build Tool: Vite
*   Styling: Tailwind CSS
*   Animations: Tailwind CSS Animate
*   Localization: i18next
*   Visual Effects: tsParticles

## 特徴

*   レスポンシブデザイン: モバイル、タブレット、デスクトップ向けに最適化されています。
*   ダイナミックな背景: モダンな視覚効果を生み出すインタラクティブなパーティクル。
*   ローカライゼーション: 複数の言語 (i18n) をサポート。
*   パフォーマンス: Viteとコンポーネントベースのアプローチによる高速なロード。
*   構造化: 読みやすさを考慮した明確なセクション分割。

## プロジェクト構造

*   Hero — メイン画面
*   Skills — 技術スキル
*   Experience — 職務経歴
*   Education — 学歴
*   Certifications — 資格
*   Projects — ポートフォリオ
*   Hobbies — 趣味
*   SelfPromotion — 自己PR

<a name="interactive-web-resume-en"></a>
# Interactive Web Resume (EN)

A modern, interactive, and fully responsive web resume built with the React + TypeScript stack. The project is developed with a focus on performance, clean code, and a pleasant user interface.

## Tech Stack

*   Framework: React 18 (Strict Mode)
*   Language: TypeScript
*   Build Tool: Vite
*   Styling: Tailwind CSS
*   Animations: Tailwind CSS Animate
*   Localization: i18next (multi-language support)
*   Visual Effects: tsParticles (interactive background)

## Features

*   Responsive Design: Optimized for mobile devices, tablets, and desktops.
*   Dynamic Background: Interactive particles creating a modern visual effect.
*   Localization: Support for multiple languages (i18n).
*   Performance: Fast loading thanks to Vite and a component-based approach.
*   Structured: Clear separation into sections for ease of reading.

## Project Structure

*   Hero — Main screen with greeting
*   Skills — Technical competencies
*   Experience — Chronological work experience
*   Education — Academic background
*   Certifications — Certificates and courses
*   Projects — Portfolio of works
*   Hobbies — Personal interests
*   SelfPromotion — Professional summary and CTA
